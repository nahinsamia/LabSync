# LabSync

A Smart Laboratory Resource & Equipment Management System developed as a DBMS Lab Project.

---

## Project Description

LabSync is a web-based laboratory management system that helps manage laboratory equipment, categories, issue records and maintenance information.

---

## Technologies Used

- HTML5
- CSS3
- Bootstrap 5
- PHP
- MySQL
- XAMPP

---

## Modules

- Dashboard
- Equipment Management
- Category Management
- Equipment Issue & Return
- Maintenance Management
